# Online Library Project

## Что это за проект

Это учебный C++17 проект онлайн-библиотеки с:

- каталогом книг
- поиском
- выдачей и возвратом книг
- очередью бронирования
- отзывами
- загрузкой и сохранением книг в файл
- интерактивным консольным меню

## Реализованные паттерны

1. Factory Method

- Точка создания выдачи книги: LoanFactory::create(...)
- Если книга доступна, создается Loan и книга блокируется
- Если книга занята, пользователь добавляется в очередь ожидания

2. Observer

- Интерфейс наблюдателя: IObserver
- User реализует IObserver
- ReservationQueue хранит список наблюдателей и отправляет уведомления через notifyAll()

Эти два паттерна удовлетворяют требованию: минимум два паттерна, один из них порождающий и не Singleton.

## Структура проекта

- include/ — все заголовочные файлы
- src/ — все исходные файлы и main.cpp
- data/books.txt — тестовые данные книг
- CMakeLists.txt — сборка через CMake

## Проверка условий задания

1. Реорганизация структуры

- Выполнено: include/, src/, data/books.txt, CMakeLists.txt

2. Загрузка и сохранение книг

- Выполнено: Catalog::loadFromFile(...), Catalog::saveToFile(...)
- Формат: title;genre;year;author
- Если файла нет, каталог стартует пустым

3. Интерактивное меню

- Выполнено: пункты 1..6 и 0 (выход с сохранением)
- Пользователь при старте ищется по имени или создается
